JavaScript Basics

Javascript is a cross-plantform, object-oriented scripting lanuage. Javascript is extremely popular for a variety of reasons. It is a small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. Javascript lives inside a host enviroment (a web broswer or node server), it can be connected to the objects of these enviroments to provide programmatic control over them.
